This is a modified Graphite theme for Cinnamon DE.
Inspired on both, Graphite Dark (colors) & Cinnamon (shapes for the shell).
Added borders for the menus, follows the blue accent color of the Graphite theme (#5294e2) or rgb(82,148,226).
